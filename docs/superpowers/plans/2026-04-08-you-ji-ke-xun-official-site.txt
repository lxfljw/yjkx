# 有迹可循官网（单页）实现计划

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** 在 `yjkx/` 内交付可部署的单页官网（8 区块、3 处「立即体验」、演示图横向浏览与点击放大、FAQ 折叠）。需要可搬运的构建物时，在 `yjkx` 内执行 `pnpm build`，输出到 **`yjkx/dist/`**（仅本目录，不碰博客仓库其他路径）。

**Architecture:** 纯静态 HTML/CSS/JS，无框架；体验链接集中在 `main.js` 的 `EXPERIENCE_URL` 常量。`yjkx/docs` 使用 `.txt` 扩展名，避免父仓库 VitePress 把设计稿当页面构建。`build-dist` 仅复制 `index.html`、`styles.css`、`main.js`、`assets/` 到 `yjkx/dist`，不拷贝 `docs/`。

**Tech Stack:** HTML5、CSS（变量 + 响应式）、原生 JS（事件委托、`<dialog>` 或自定义 lightbox）。
